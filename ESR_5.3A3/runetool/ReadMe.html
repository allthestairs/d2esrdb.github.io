Runetool Readme

PREFACE:
  • This program was created by Smiiiff for ES 2.2 and updated for ES 3.00 R by Stabby McTwist and tsuru.
  • Though there are some known issues, the sorce is lost and the program can no longer be updated.
    Please see NOTES section below.
USAGE:
  • Select the runes you have in the rune panel.
    You can save your rune selection by "Files - Save".
    You can load your rune selection by "Files - Load".
    You can select all runes by "Files - Load - All".
    You can unselect all runes by "Revert" button.
  • Select the base items and socket numbers by the checkbox.
  • Hit "Search" button.
  • Click each listed runeword to see the stats.
NOTES:
  • "Other" checkbox now searches Charms and Quivers.
  • "Runename *" in the list means that runeword accepts (or requires) an extra Jewel.
    Runetool can't search the runeword with a jewel by the socket number correctly.
    For example, RW Stone with an extra jewel is for "3 socket", but is only hit when searched by "2 socket".
    All the runewords with a jewel are for Body Armor and Shields.
    When you search Body Armor/Shield runewords, please also check the words with one less socket and an asterisk.
  • The saved rune selections are lost when the runeword data is updated.
    This is because the selection and the runeword data are stored in the same file.
  • This program doesn't suooprt gemwords, Kanji runewords and LoD runewords.
    Please see ES databasethe and Arreat Summit for them.
